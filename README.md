# World Happiness Report Analysis

## Project Description

This project analyzes the 2016 World Happiness dataset using Python.

## Technologies

- Python
- Pandas
- Matplotlib
- Plotly

## Features

- Data Cleaning
- Exploratory Data Analysis
- Interactive Dashboard
- Maps
- Scatter Plots
- Pie Charts

## Results

Countries with higher GDP generally have higher happiness scores.

## Dataset

World Happiness Report 2016

## How to Run

pip install -r requirements.txt

python analysis.py
